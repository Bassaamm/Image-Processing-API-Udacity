## Setup to run the project

### Install Dependencies

    $ npm i 

### Run the project

    $ npm run start 

### Build the project

    $ npm run build

### test the project

    $ npm run test
### Run jasmine

    $ npm run jasmine

### Run eslint

    $ npm run eslint

### Run Prettier

    $ npm run prettier
